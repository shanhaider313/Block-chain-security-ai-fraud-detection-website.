# Block-chain-security-AI-Powered Blockchain Fraud Detection Website.

A smart web application that uses artificial intelligence to detect and prevent fraudulent activities. Built with a Python backend (Flask) and a responsive front-end, it analyzes user input and transaction patterns to identify potential fraud in real-time. Ideal for learning Al integration with web development.

## Features
- AI-powered fraud detection
- Flask backend with Python
- Responsive front-end
- Real-time transaction analysis
- Easy to extend and integrate
